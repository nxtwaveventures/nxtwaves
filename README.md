# NxtWaves Innovation Hub

A high-end, immersive landing page for 'nxtwaves.in', inspired by the Google DeepMind Veo showcase.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Key Features

- **Hero Section:** Full-screen immersive background with magnetic CTA.
- **Video Reveal:** Scroll-linked animation expanding video container.
- **Builds Gallery:** Interactive grid with hover effects.
- **Glassmorphism UI:** Modern, translucent design elements.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open [http://localhost:3000](http://localhost:3000)** with your browser to see the result.

## Customization

- Update images/videos in `src/components/Hero.tsx` and `src/components/VideoReveal.tsx`.
- Modify specific tools in `src/components/BuildsGallery.tsx`.
- Adjust colors in `src/app/globals.css`.
